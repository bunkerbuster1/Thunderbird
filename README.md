5. I know there is alot of stuff but just click on terminal if it doesnt automatically get you on the terminal
6. run npm install in the terminal
7. if it says to update something using a command just run the command and then follow my steps
8. run npm start
9. a port should start running, just click on the ports section next to terminal and open that link!
